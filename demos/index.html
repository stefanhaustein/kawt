kAWT - Samples and Screenshots

KawtDemo

Palm Desktop Applet* MIDP

Description

The KawtDemo shows most of the kAWT widgets currently available like Panels, Buttons, Checkboxes, Textfields etc.

*) If your browser cannot display the applet, please use the appletviewer from SUN:

   appletviewer http://www.kawt.de/demos/index.html

ColorDemo

Palm 4 bit Palm 8 bit Prism 16 bit
Desktop MIDP

The ColorDemo shows the usage of color with kAWT. In order to actually see colors on a Palm IIIc, the ColorKVM is required.

Download

  • Source: ColorDemo.java
  • Palm Binary: ColorDemo_prc.zip
  • Desktop jar file: ColorDemo.jar
    java -jar ColorDemo.jar

PenDemo

Download

KVM 1.0		 GifDemo_prc.zip Unzip the file 'GifDemo_prc.zip' and install the prc-file using Palm Desktop installation tool, or just drag the file on POSE.

Changes in the POSE preferences required for Jbed

The Jbed compiler makes some direct access to Palm hardware that will cause warings in POSE. Please change the "Palm OS Emulator.ini" file in your Windows root directory to turn of the corresponding warning messages:

  • ReportHardwareRegisterAccess=0
  • ReportLowMemoryAccess=0
  • ReportStackAlmostOverflow=0
  • ReportStackOverflow=0
