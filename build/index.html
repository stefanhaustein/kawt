Misc. tools and patches:

  • ByteArrayDumper.java: Dumps a gif as bytearray to the shell (full package name to include this file: de.trantor.misc).
  • Protocol.java: The comm-protocol for the Generic Connection Framework on the J2SE. Java Comm-API must be installed properly ! (full package name to include this file: com.sun.cldc.io.j2se.comm).
  • Socket.java: java.net.Socket, wrapping the javax.microedition.io Connector etc. for JVM-Like Socket support.
