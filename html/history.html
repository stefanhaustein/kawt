23. Jul. 2000   Handheld Express is available on http://www.borland.com/jbuilder/hhe/ for use with JBuilder3.5. Handheld Express is a poweful tool that is able to create Palm prc-files for J2ME CLDC applications directly from the JBuilder IDE. With Handheld Express, it is no longer necessary to write complicated makefiles. Compilation, preverification and the palm.database.MakePalmApp call are performed automatically by a Wizard. Just select 'Deploy to PalmOS...' from the JBuilder Wizard menu. Currently, we are going to use Handheld Express to build our KawtSample. We hope that full support for the kAWT widgets will be available soon.
