The xKVM / Color KVM - Download

At this time, two versions of the xKVM are avaliable, one including a "romized" version of kAWT v0.9951 and a second one without the kAWT classes.

When you download the xKVM, please keep in mind, that this modified KVM version is for research and evaluation purposes only and not for commercail use.

Our xKVM is a feasibility study to show that it is possible to add device specific features to SUN's KVM.

The complete SCSL can be downloaded here :http://sun.com/software/communitysource

By downloading the xKVM / ColorKVM you agree to the license terms of the SCSL!!!

xKVM

  • xKVM.zip: The xKVM (Build 1 6-19-2001)
  • xKVM_ROMIZED.zip: The xKVM (Build 3 11-12-2001) with kAWT v1.0 ROMized.

ColorKVM (based on CLDC 1.0, no longer supported)

  • ColorKVM.zip: The ColorKVM (Build 9, multi PDB) without the ROMized kAWT classes with support for 16 Bit colordepth on VISOR Prism.
  • ColorKVM_kAWTROMizedv09903.zip: The ColorKVM (Build 10, multi PDB) with a ROMized version of kAWT v0.9903 with support for 16 Bit colordepth on VISOR Prism. It is not necessary to add the kawt.pdb file to run the KawtSample or any other kAWT application.
